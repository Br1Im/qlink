# Создание тестового пользователя

## Быстрый старт

### Windows
```cmd
scripts\create-test-user.bat
```

### Linux/Mac
```bash
chmod +x scripts/create-test-user.sh
./scripts/create-test-user.sh
```

### Напрямую через Docker
```bash
docker exec qlink-backend node /app/packages/backend/scripts/create-test-user.js
```

## Данные для входа

После выполнения скрипта вы сможете войти в систему:

- **Email**: `test@qlink.tech`
- **Пароль**: `Test123456`
- **Роль**: Владелец бизнеса
- **Бизнес**: Тестовый Салон (с 3 услугами)

## Что создается

Скрипт автоматически создает:

1. **Пользователя** с ролью владельца бизнеса
2. **Бизнес** - "Тестовый Салон" с полной информацией
3. **Услуги**:
   - Стрижка женская (2500₽, 60 мин)
   - Окрашивание волос (4500₽, 120 мин)
   - Маникюр (1800₽, 90 мин)

## Требования

- Docker контейнеры должны быть запущены
- Backend контейнер должен быть доступен
- База данных должна быть инициализирована

## Примечание

Если пользователь уже существует, скрипт просто выведет данные для входа без создания дубликата.
